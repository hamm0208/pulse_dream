# Pulse Dream (Porsche models rental)

## Note:
I had more plans for this website, such as implementing 3D cars, better UI for the fleets, however due to time constraints i was not able to achieve the desired result

## Description
This is a website project built with mainly HTML, CSS, JavaScript, VueJS, Bootstrap. This guide will help you set up and run the project on your local machine.

GITHUB: [Pulse Dream](https://github.com/hamm0208/pulse_dream)

## Prerequisites
Before you begin, ensure you have met the following requirements:
- You are using one of the following operating systems: Windows, macOS, or Linux.
- You have installed the following software:
  - **XAMPP**: If you have downloaded xampp you can skip the 3-4 because Apache and MySQL is already on XAMPP
  - **Apache**: A web server to serve your website.'
  - **GitHub Desktop**: To easily install the website'
    

## Installation
1. On the GitHub Desktop application, File -> Clone Repository
2. Change the type to "URL"
3. Copy and paste this into the URL section
```bash
https://github.com/hamm0208/pulse_dream
```
4. Change the local path to the XAMPP's htdocs directory: 
```bash
C:\xampp\htdocs
```
 
## Running the Project
1. Open Your Web Browser:
   
Navigate to http://localhost/pulse_dream to see the website.
Note: Ensure that the default folder name is pulse_dream. If your folder has a different name, make sure to change it in the directory to match exactly.

**Steps to Ensure Proper Setup**
1. Start XAMPP:

Open XAMPP and start the Apache server.
If your project uses a database, also start the MySQL server.

2. Check Your Project Directory:

Ensure that your project files are placed inside the htdocs folder in the XAMPP directory. For example, if you installed XAMPP in C:\xampp, your project should be in C:\xampp\htdocs\pulse_dream.

3. Folder Name Consistency:

Make sure the folder containing your project is named pulse_dream. If it is named differently, update the folder name to pulse_dream or change the URL accordingly (e.g., http://localhost/your_folder_name).

4. Navigate to Your Project:

Open a web browser (e.g., Chrome, Firefox, Edge).
Enter http://localhost/pulse_dream in the address bar and press Enter.

5. Verify the Website:

You should see your website's home page if everything is set up correctly. If you encounter any issues, check the following:

The Apache server is running.

Your project files are correctly placed in the htdocs directory.

The folder name matches the URL path.
